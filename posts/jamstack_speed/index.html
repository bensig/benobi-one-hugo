Lightning fast performance with JAMstack

This website has been built with serverless architecture using JAMstack.

The speed of this architecture is really unparalleled. The trick is that I am delivering static content on CDNs.

The entire source code for this site is hosted on GitHub. I have written a previous post about building this site.

Blazing Speed

Take a look at the blazing performance of https://benobi.one - the site usually gets a score of 95-100 in terms of speed. You can test this against any other site using that some Google tool.

The reason that site speed is important is not just because Google says so - but also that if your site takes more than 3 seconds to load, 25% of users will click away.

I have not seen a faster architecture than this and I would suggest that it becomes a new standard for sites in the future.

I definitely plan to use this architecture in the future where it makes sense.
